Industrial AI

Контур промышленной аналитики для добычи

Задача: единый слой KPI и отклонений по участкам без ручных сводок.
Решение: industrial AI-пайплайн: ETL из MES/SCADA, правила + ML-скоринг аномалий, дашборд для диспетчеров.
Результат: более раннее выявление отклонений, прозрачные сменные отчёты, меньше простоев на критичных линиях.
Стек: Python, PostgreSQL, Power BI, Docker, API оркестрации.

AI agents

Orchestration документов и заявок

Задача: снять ручной разбор входящих актов и заявок в ERP.
Решение: AI-агенты: извлечение полей, валидация, маршрутизация в workflow (n8n/Make), human-in-loop на исключениях.

Power BI · KPI

Аналитические системы и дашборды цеха

Задача: согласованные производственные дашборды и отчётность для руководства и смен.
Решение: модель данных + Power BI: OEE, простои, scrap, сравнение смен; публикация в едином каталоге аналитических систем.
Результат: один источник правды по KPI, меньше Excel-рассылок, быстрее разбор инцидентов.
Стек: Power BI, DAX, SQL Server/PostgreSQL, gateway, при необходимости Python ETL.
